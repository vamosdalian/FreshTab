📋 默认配置说明

第一次打开 FreshTab 时，您会看到：

  • 🌟 常用网站 - 一个默认分组，包含6个预设标签
  • Google - 使用搜索 emoji 🔍
  • GitHub - 使用猫咪 emoji 🐱
  • 知乎 - 使用文字图标 "知"
  • 微博 - 使用记事本 emoji 📝
  • YouTube - 使用电视 emoji 📺
  • Netflix - 使用电影 emoji 🎬

🎨 图标类型演示：

  • Emoji - 丰富的表情符号选择，无需网络加载
  • 文字 - 自定义1-2个字符作为图标
  • 网站图标 - 可选自动从多个服务获取网站的真实 favicon

✨ 用户可以：

  • 点击 "+" 添加新标签
  • 编辑现有标签的名称、URL、图标和颜色
  • 删除不需要的标签
  • 重新排序标签
  • 创建新的分组来整理标签
